# Sistema de Reservas de Tutorías para Colegio Público

Sistema web dinámico para reservar tutorías con profesores, desarrollado con Node.js y Express. Cumple con la normativa RGPD de protección de datos.

## Características

✅ **Reservas dinámicas** - Horarios disponibles por profesor
✅ **RGPD Compliant** - Datos temporales, consentimiento explícito
✅ **Sin almacenamiento permanente** - Datos eliminados tras 24h
✅ **Confirmaciones por email** - Con PDF adjunto
✅ **Interfaz moderna** - Diseño responsive y accesible
✅ **Independiente** - No requiere Microsoft 365 ni Google Workspace

## Requisitos

- Node.js ≥ 18
- NPM ≥ 9
- Navegador web moderno
